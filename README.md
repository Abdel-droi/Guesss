# Guesss : A number guessing game

A fun and interactive number guessing game built with Python and Tkinter. Each round has a unique twist that keeps the game fresh and challenging !

# How to Play:

Launch the game
Complete 3 rounds each with a different challenge
Try to guess the correct number as fast as possible
Your time and attempts are tracked every round
At the end, see your total score!

# Rounds:
Round 1: Guess any number
Round 2 :Guess an even number
Round 3 : Guess a prime number

# Requirements
Python 3.14
Tkinter (built into Pycharm, no install needed)

# Built With
Python  — Core logic
Tkinter — GUI interface
time — Speed tracking
random — Number generation
